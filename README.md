# EasyMaths

EasyMaths 是一个数学笔记与练习整理仓库，用 VitePress 发布为可浏览的文档站点。

## 内容

- [浅谈三角函数](三角函数.md)
- [神卷](神卷.md)
- [mdr卷](mdr卷.md)

## 本地预览

```powershell
pnpm install
pnpm docs:dev
```
